## How to Update

When making changes:

1. Add entry under `[Unreleased]` section
2. Use categories: `Added`, `Changed`, `Deprecated`, `Removed`, `Fixed`, `Security`
3. On release, move `[Unreleased]` to new version with date

Example:
```markdown
## [1.1.0] - 2024-02-15

### Added
- New feature X

### Fixed
- Bug in Y
```
